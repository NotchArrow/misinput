A simple clientside mod that allows you to setup a list of players that you do not want to attack. If you attempt to hit a player in the list that you configure, then the mod will cancel the action and prevent you from hitting them. The list of usernames will save across sessions, servers, and worlds. It's easily configurable with commands to add or remove players.

`/misinput add (username)` - Add a username to the list

![Misinput Add](https://cdn.modrinth.com/data/cached_images/29f3f59f5ce842d68c57a793c2e5da4388a130db.png)

`/misinput list` - List all of the usernames alphabetically

![Misinput List](https://cdn.modrinth.com/data/cached_images/338421c02d3ad36e262c7edc6f754d54057e5d5b.png)

`/misinput remove (username)` - Remove a username from the list

![Misinput Remove](https://cdn.modrinth.com/data/cached_images/28aa4b1bdf6d0c20f6258cf893d42c809c336593.png)
